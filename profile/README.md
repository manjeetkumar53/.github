A curated engineering portfolio by [Manjeet Kumar](https://github.com/manjeetsingh53), focused on production-oriented GenAI systems, backend architecture, reliability, evaluation, observability, and safe automation.

This profile is organized as a reviewable portfolio, not a random repository list. Start with the featured AI systems, then use the catalog below to inspect earlier backend, cloud, and AI experiments.

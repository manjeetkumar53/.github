# Manjeet Kumar AI Engineering Portfolio

A focused collection of production-oriented AI engineering systems by [Manjeet Kumar](https://github.com/manjeetsingh53).

This portfolio is built around practical GenAI architecture: model routing, agent orchestration, RAG evaluation, human-in-the-loop automation, observability, safety controls, and production trade-offs.

## Architecture Themes

- **Cost control:** route requests by complexity and compare against baselines.
- **Reliability:** use retry, circuit breaker, fallback, and deterministic local providers.
- **Safety:** enforce guardrails, allowlists, RBAC, and approval gates.
- **Evaluation:** measure routing accuracy, planner behavior, retrieval recall, and faithfulness.
- **Observability:** expose request IDs, metrics endpoints, dashboards, event stores, and audit trails.
